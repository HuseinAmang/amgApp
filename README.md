## Tentang amgApp
Node.js Express Framework Rest API dengan JWT

## Installasi 
Node install

## Start
Node server.js

## API User
###### User SignUp
POST:: /api/auth/signup

###### User SignIn
POST:: /api/auth/signin

###### User Profile
GET:: /api/user/:id

## API Post
###### Create Post
POST:: /api/post/

###### Read All Post
GET:: /api/post/

###### Read All Post Published
GET:: /api/post/published

###### Read Post by Id
GET:: /api/post/:id

###### Update Post by Id
PUT:: /api/post/:id

###### Delete Post by Id
DELETE:: /api/post/:id

###### delete all
DELETE:: /api/post/
